# StudyRec

学習時間を記録・可視化する個人開発Webアプリです。

## 概要
React + TypeScript の学習を目的に制作。
学習の継続を助けるため、記録と可視化にフォーカスしています。

## 使用技術
- React
- TypeScript
- Vite
- CSS Modules
- ApexCharts

## 機能
- 学習記録の追加
- 科目別・日別・月別の集計
- グラフによる可視化
