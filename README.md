# StudyReco
学習の継続を目的に、学習データの構造設計と可視化に重点を置いて開発した個人開発Webアプリです。
あわせて、日常的に使われることを想定し、操作が直感的でシンプルなUI設計を意識しています。

## 概要
React + TypeScript を用いた個人開発Webアプリ。
学習の継続を支援するため、データ構造と可視化を意識し、日常的に使われることを想定して設計しています。

## 使用技術
- React
- TypeScript
- Vite
- CSS Modules
- ApexCharts

## 機能
- 学習記録の追加  
  └ 日付・科目・学習時間を分離して管理し、集計や分析を前提としたデータ構造で設計

- 学習データの集計・可視化  
  └ 集計ロジックを共通化し、一覧表示とグラフ表示で同じデータを再利用できるよう設計

- 月間学習ランキングモデル  
  └ 1ヶ月分の学習時間を集計し、学習時間の多い科目順に順位付けできる構成


## 今後追加したい機能
- ローカルストレージ / Firebase を用いたデータ永続化  
  └ ブラウザ更新や端末変更後も学習データを保持できるようにするため

- ユーザーごとの学習記録管理（認証）  
  └ 個人単位でデータを管理し、将来的な機能拡張を見据えた設計

- 週別・期間指定での学習時間分析  
  └ 学習傾向を把握しやすくするため、集計単位を柔軟に切り替えられるようにする

- タイムウォッチ機能  
  └ 学習開始・終了を計測し、入力負荷を減らしながら正確な学習時間を記録する

- 学習記録のシェア機能  
  └ 学習状況を外部に共有できる仕組みを設け、継続学習のモチベーション向上を意識
